## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

Install dependencies:

```bash
cd weather-app
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser at:

http://localhost:5173
